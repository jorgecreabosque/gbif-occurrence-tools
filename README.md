# GBIF Occurrence Tools

Herramientas en Python para consultar registros de ocurrencia de especies desde la API de GBIF y generar datasets listos para SIG.

## Descripción

Este script permite descargar registros de ocurrencia desde GBIF utilizando paginación automática y generar archivos listos para su uso en SIG.

Los resultados se exportan a formatos comúnmente utilizados en análisis espaciales.

## Funcionalidades

- Consulta automática a la API de GBIF
- Descarga paginada de registros
- Exportación a CSV
- Exportación a GeoPackage (GPKG)
- Exportación a KMZ
- Generación de tablas resumen
